1. Select the `EasyResolutions_Settings` asset placed alongside this `Readme` file and click `Add to Preloaded Assets`.
2. Remove any scenes from the `Scene List` in `File -> Build Profiles`,
3. Add `Packages/com.vklubkov.easyresolutions/Assets/ZeroScene` to the `Scene List` in `File -> Build Profiles`, so its index is 0.
4. Add scenes from the `Scenes` folder of this sample to the `Scene List` in `File -> Build Profiles`, make sure `ZeroScene` index is 0.
5. Go to `EasyResolutions_Settings` asset and make sure `Zero Scene` path is `Packages/com.vklubkov.easyresolutions/Assets/ZeroScene`, and the`Starting Scene (group)` is a group ending with `StartingScene`.
